# 🎭 Impostor - Das Spiel

Ein Party-Spiel für 3-20 Spieler, bei dem ihr den Imposter entlarven müsst!

## Spielanleitung

### Vorbereitung
- Alle Spieler erhalten ein geheimes Wort
- Ein Spieler ist der **Imposter** und erhält nur die Kategorie

### Spielablauf
1. **Begriffe ansehen** - Smartphone wird reihum gegeben, jeder sieht verdeckt seinen Begriff
2. **Hinweise geben** - Jeder Spieler gibt mündlich einen Hinweis (ein Wort)
3. **Diskutieren** - Besprecht, wer der Imposter sein könnte
4. **Abstimmen** - Wählt den Imposter

### Gewinnen
- **Gruppe gewinnt** wenn der Imposter entlarvt wird
- **Imposter gewinnt** wenn er nicht entdeckt wird
